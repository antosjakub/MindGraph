# MindGraph





| Key                 | Action                       |
| ------------------- | ---------------------------- |
| N                   | New node                     |
| C                   | Connect mode                 |
| G                   | Group selected               |
| B                   | Toggle boards panel          |
| F                   | Fit view                     |
| Enter               | Open modal for selected node |
| Delete              | Delete selected              |
| Esc                 | Cancel / close               |
| + / -               | Zoom in/out                  |
| Arrows              | Pan the view                 |
| Ctrl+A              | Select all                   |
| Shift+click         | Multi-select                 |
| Shift+drag          | Rectangle select             |
| Double-click canvas | Create node at cursor        |
